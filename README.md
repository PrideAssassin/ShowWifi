# ShowWifi

## 一、简介：
-    1.1 ShowWifi是一款能够查看本机所有连过的wifi的密码。

## 二、运行环境
-   2.1 仅支持windows
-   2.2 台式机一般都不自带无线网卡，如果在台式机上插入外置无线网卡，需要自行测试。

## 三、原理：
-   3.1 使用cmd命令查询连接电脑中的配置文件。

## 四、bug待完善：

      - 1） 当前用户配置文件
      - 2）所有用户配置文件
